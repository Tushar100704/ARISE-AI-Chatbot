# 🤖 AI Chatbot using Groq + Next.js

This project is a real-time AI chatbot built using **Next.js, Vercel AI SDK, and Groq API**.  
It supports fast streaming responses and can be extended with tools and memory.

---

## 🚀 Features

- ⚡ Ultra-fast AI responses using Groq API  
- 💬 Real-time streaming chat interface  
- 🧠 Context-aware responses (extendable with memory)  
- 🛠️ Tool integration support (weather, APIs, etc.)  
- 🎨 Modern UI using Tailwind + shadcn/ui  

---

## 🧠 Tech Stack

- Frontend: Next.js (App Router)  
- AI Integration: Vercel AI SDK + Groq API  
- UI: Tailwind CSS + shadcn/ui  
- Deployment: Vercel  

---

## ⚙️ How It Works

1. User sends a message  
2. Request goes to backend API route  
3. Groq model processes input  
4. Streaming response is returned in real-time
